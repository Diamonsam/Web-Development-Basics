→ WHAT IS CSS?

  • CSS Stands for Cascading Style Sheets which is style sheet language that used to handle the presentation of the web page containing HTML.
  • CSS Gives styles to raw HTML.
  • CSS is used to give style to our web pages.
  • CSS is used to make website responsive.
  • HTML is used to strucure a website.
  • CSS takes resposibility of design. It makes our website beautiful and modern looking.

→ THREE WAYS YOU CAN ADD CSS to HTML

  1. Inline CSS : Inline CSS is added to the elements directly using style attribute. This CSS is will execute over other two CSS if defined.
    For Example :- <p style="color:red;">

  2. Internal CSS : Internal CSS is kept inside head tags in <style> tags in HTML. This CSS is will execute over external css if defined after link tag.
    For Example :-
    <head>
       <style>
          p{
             color:red;
          }
       </style>
    </head>

  3. External CSS : External CSS is kept separatelt inside a css style sheet file. & linked to html in head tags. If no Inline or Internal CSS is define for element than this CSS will take effect.
    For Example :- <link rel="stylesheet" href="/style.css">
    Here style.css is new externalfile which is linked to html with link tag.

→ WHAT IS A CSS SELECTOR?

  • CSS Selectors are used to find the element whose prperty will be set.
  • Selectors are used to target the HTML Elements.
  • Selectors makes it easy for us to easily target single/multiple HTML Elements in markup.
SELECTOR PROPERTY VALUE
p {color: red;}

→ TYPES OF BASIC CSS SELECTOR?

  1. CSS Univarsal Selector : CSS Univarsal Selector is used to selects all elements of HTML document.
    For Example :- Select all elements of Entire HTML Document & set margin to 0 pixel:
    Syntax:-
       <style>
          *{
             margin:0;
          }
       </style>

  2. CSS Specific Univarsal Selector : The * selector can also select all elements inside another element.
    For Example :- Select all elements inside <div> elements and set their background color to yellow:
    Syntax:-
       <style>
          div *{
             background-color:yellow;
          }
       </style>

  3. CSS Element Selector : CSS Element Selector is used with reference to Tag Name to change the style of that particular Tag.
    For Example :- Select all <p> elements and set thier text color to red:
    Syntax:-
       <style>
          p{
             color:red;
          }
       </style>

  4. CSS Id Selector : CSS Id Selector is used with reference to Tag Id to change the style of that particular Tag.
    For Example :- Select unique element with <p id="para1"> set thier text color to red:
    Syntax:-
       <style>
          #para1{
             color:red;
          }
       </style>

  5. CSS Class Selector : CSS Class Selector is used with reference to Tag Class to change the style of that particular Tag.
    For Example :- Select all elements with <div class="container"> set thier text color to red:
    Syntax:-
       <style>
          .container{
             color:red;
          }
       </style>

  6. CSS Grouping Selector : CSS Grouping Selector is used with reference to Multiple Tag Names, Ids Or Classses to change the style of that particular Tag.
    For Example :- Select multiple elements with tag name/id/class like <p>,<div class="container">,<span id="span1"> and set thier text color to red:
    Syntax:-
       <style>
          p, .container, #span1{
             color:red;
          }
       </style>

→ CSS COLORS

→ Colors in CSS can be specified by the following methods:
  1. Hexadecimal colors :-
    • A hexadecimal color is specified with: #RRGGBB, where the RR (red), GG (green) and BB (blue) hexadecimal integers specify the components of the color. All values must be between 00 and FF.
    • For example, the #0000ff value is rendered as blue, because the blue component is set to its highest value (ff) and the others are set to 00.
    • Syntax:-
      1. #p1 {background-color: #ff0000;} /* red */
      2. #p2 {background-color: #00ff00;} /* green */
      3. #p3 {background-color: #0000ff;} /* blue */
  2. Hexadecimal colors with transparency :-
    • A hexadecimal color is specified with: #RRGGBB. To add transparency, add two additional digits between 00 and FF.
    • Syntax:-
      1. #p1a {background-color: #ff000080;} /* red transparency */
      2. #p2a {background-color: #00ff0080;} /* green transparency */
      3. #p3a {background-color: #0000ff80;} /* blue transparency */
  3. RGB colors :-
    • An RGB color value is specified with the rgb() function, which stands for rgb(red, green, blue).
    • Each parameter (red, green, and blue) defines the intensity of the color and can be an integer between 0 and 255 or a percentage value (from 0% to 100%).
    • For example, the rgb(0,0,255) value is rendered as blue, because the blue parameter is set to its highest value (255) and the others are set to 0.
    • Also, the following values define equal color: rgb(0,0,255) and rgb(0%,0%,100%).
    • Syntax:-
      1. #p1 {background-color: rgb(255, 0, 0);} /* red with opacity */
      2. #p2 {background-color: rgb(0, 255, 0);} /* green with opacity */
      3. #p3 {background-color: rgb(0, 0, 255);} /* blue with opacity */
  4. RGBA colors :-
    • RGBA color values are an extension of RGB color values with an alpha channel - which specifies the opacity of the object.
    • An RGBA color is specified with the rgba() function, which stands for rgba(red, green, blue, alpha).
    • The alpha parameter is a number between 0.0 (fully transparent) and 1.0 (fully opaque).
    • Syntax:-
      1. #p1 {background-color: rgba(255, 0, 0, 0.3);} /* red with opacity */
      2. #p2 {background-color: rgba(0, 255, 0, 0.3);} /* green with opacity */
      3. #p3 {background-color: rgba(0, 0, 255, 0.3);} /* blue with opacity */
  5. HSL colors :-
    • HSL stands for hue, saturation, and lightness - and represents a cylindrical-coordinate representation of colors.
    • An HSL color value is specified with the hsl() function, which stands for hsl(hue, saturation, lightness).
    • Hue is a degree on the color wheel (from 0 to 360) - 0 (or 360) is red, 120 is green, 240 is blue.
    • Saturation is a percentage value; 0% means a shade of gray and 100% is the full color.
    • Lightness is also a percentage; 0% is black, 100% is white.
    • Syntax:-
      1. #p1 {background-color: hsl(120, 100%, 50%);} /* green */
      2. #p2 {background-color: hsl(120, 100%, 75%);} /* light green */
      3. #p3 {background-color: hsl(120, 100%, 25%);} /* dark green */
      4. #p4 {background-color: hsl(120, 60%, 70%);} /* pastel green */
  6. HSLA colors :-
    • HSLA color values are an extension of HSL color values with an alpha channel - which specifies the opacity of the object.
    • An HSLA color value is specified with the hsla() function, which stands for hsla(hue, saturation, lightness, alpha).
    • The alpha parameter is a number between 0.0 (fully transparent) and 1.0 (fully opaque).
    • Syntax:-
      1. #p1 {background-color: hsla(120, 100%, 50%, 0.3);} /* green with opacity*/
      2. #p2 {background-color: hsla(120, 100%, 75%, 0.3);} /* light green with opacity*/
      3. #p3 {background-color: hsla(120, 100%, 25%, 0.3);} /* dark green with opacity*/
      4. #p4 {background-color: hsla(120, 60%, 70%, 0.3);} /* pastel green with opacity*/
  7. Predefined/Cross-browser color names :-
    • 140 color names are predefined in the HTML and CSS color specification.
    • For example: blue, red, coral, brown, etc:
    • Syntax:-
      1. #p1 {background-color: blue;}
      2. #p2 {background-color: red;}
      3. #p3 {background-color: coral;}
      4. #p4 {background-color: brown;}
  8. With the currentcolor keyword :-
    • The currentcolor keyword refers to the value of the color property of an element.
    • Syntax:- 
                                  #myDIV 
                            {
                            
                                color: blue; /* Blue text color */
                                border: 10px solid currentcolor; /* Blue border color */
                            
                            }

→ CSS BACKGROUND

  1. Background-color Attribute
    • The CSS background-color property defines the background color for an HTML element.
    • Syntax:-
      1. #div1 {background-color: blue;} In addition to background-color this element width is 500px & height is 50px
      2. #p2 {background-color: red;}In addition to background-color this element width is 300px & height is 70px
      3. #span {background-color: green;}In addition to background-color this element width is 1000px & height is 40px
  2. Background-color Attribute With Opacity
    • The opacity property specifies the opacity/transparency of an element. It can take a value from 0.0 - 1.0. The lower value, the more transparent:
    • You can use Opacity attribute with any elements like div, p, span etc.
    • Syntax:-
      1. #div1 {background-color: blue; opacity:0.9;} Background color with 90% opacity/transparency.
      2. #p2 {background-color: blue; opacity:0.6;} Background color with 60% opacity/transparency.
      3. #span {background-color: blue; opacity:0.3;} Background color with 30% opacity/transparency.
      4. #span {background-color: blue; opacity:0.1;} Background color with 10% opacity/transparency.
  3. Background-Image Attribute
    • The background-image property specifies an image to use as the background of an element.
    • By default, the image is repeated so it covers the entire element.
    • Syntax:-

      div{background-image: url(/Images/Self.png);}

  4. Background-Repeat Attribute
    1. NO-REPEAT
      • By default, the background-image property repeats an image both horizontally and vertically.
      • To stop it from repeating we can use background-repeat attribute
      • Syntax:-

        div{background-image: url(/Images/Self.png); background-repeat: no-repeat;}



    2. REPEAT VERTICALLY
      • To repeat an image vertically, set background-repeat: repeat-y;
      • Syntax:-

        div{background-image: url(/Images/Self.png); background-repeat: repeat-y;}



    3. REPEAT HORIZONTALLY
      • To repeat an image horizontally, set background-repeat: repeat-x;
      • Syntax:-

        div{background-image: url(/Images/Self.png); background-repeat: repeat-x;}

  5. Background-Position Attribute
    1. LEFT-TOP
      • The background-position property sets the starting position of a background image.
      • By default, background-position is left-top
      • Syntax:-

        div{background-position: left top;}



    2. LEFT-CENTER
      • To set background image in Left side in center position
      • Syntax:-

        div{background-position: left center;}



    3. LEFT-BOTTOM
      • To set background image in Left side in bottom position
      • Syntax:-

        div{background-position: left bottom;}



    4. CENTER-TOP
      • To set background image in Center side in Top position
      • Syntax:-

        div{background-position: center top;}



    5. CENTER-CENTER
      • To set background image in Center side in center position
      • Syntax:-

        div{background-position: center center;}



    6. CENTER-BOTTOM
      • To set background image in Center side in bottom position
      • Syntax:-

        div{background-position: center bottom;}



    7. RIGHT-TOP
      • To set background image in Right side in top position
      • Syntax:-

        div{background-position: right top;}



    8. RIGHT-CENTER
      • To set background image in Right side in center position
      • Syntax:-

        div{background-position: right center;}



    9. RIGHT-BOTTOM
      • To set background image in Right side in bottom position
      • Syntax:-

        div{background-position: right bottom;}



    10. USING PERCENTAGE / VALUES
      • You can also set value using percentage value.
      • The first value is the horizontal position and the second value is the vertical.
      • The top left corner is 0% 0%.
      • The right bottom corner is 100% 100%.
      • If you only specify one value, the other value will be 50%.
      • Default value is: 0% 0%
      • Syntax:-

        div{background-position: 20% 30%;}

        div{background-position: 200px 200px;}



  6. Background-attachment Attribute
    • The background-attachment property specifies whether the background image should scroll or be fixed (will not scroll with the rest of the page):
    • Syntax:-

      div{background-attachment: fixed;}

  10. Background-size Attribute
    • The background-size property specifies the size of the background images.
    • There are four different syntaxes you can use with this property:
        1. the keyword syntax ("auto", "cover" and "contain"),
        2. the one-value syntax (sets the width of the image (height becomes "auto"),
        3. the two-value syntax (first value: width of the image, second value: height),
        4. the multiple background syntax (separated with comma).
      1. Auto:- Default value. The background image is displayed in its original size.

        Syntax:-

        div{background-size: auto;}



      2. Cover:- Resize the background image to cover the entire container, even if it has to stretch the image or cut a little bit off one of the edges.

        Syntax:-

        div{background-size: cover;}



      3. Contain:- Resize the background image to make sure the image is fully visible

        Syntax:-

        div{background-size: contain;}



      4. One-Value:- Sets the width and height of the background image to same value / Sets the width and height of the background image in percent of the parent element when using percentage.

        Syntax:-

        div{background-size: 200px;}



      5. Two-Value:- Sets the width and height of the background image. The first value sets the width, the second value sets the height.

        Syntax:-

        div{background-size: 200px 400px;}



→ CSS BOX MODEL

  1. Element-width Attribute:-

    • In order to set the width of an element correctly in all browsers, you need to know how the box model works.
    • When you set the width property of an element with CSS, you just set the width of the content area.
    • To calculate the full size of an element, you must also add padding, borders and margins.
    • The total width of an element should be calculated like this:
    • Total element width = width + left padding + right padding + left border + right border + left margin + right margin


  2. Element-height Attribute:-

    • In order to set the height of an element correctly in all browsers, you need to know how the box model works.
    • When you set the height property of an element with CSS, you just set the height of the content area.
    • To calculate the full size of an element, you must also add padding, borders and margins.
    • The total height of an element should be calculated like this:
    • Total element height = height + top padding + bottom padding + top border + bottom border + top margin + bottom margin


  3. For Example:-

       <style>
          div{
             width: 320px;
             height:420px;
             padding: 10px;
             border: 5px solid gray;
             margin: 0;
          }
       </style>

    ←←←←←←This <div> element will have a total width of 350px:
          Here is the calculation:
          320px (width) + 20px (left + right padding)+ 10px (left + right border)+ 0px (left + right margin) = 350px

    ←←←←←←This <div> element will have a total height of 450px:
          Here is the calculation:
          420px (height) + 20px (top + bottom padding)+ 10px (top + bottom border)+ 0px (top + bottom margin) = 450px

→ CSS MARGINS

  1. Margin Attributes:-
    • The CSS margin properties are used to create space around elements, outside of any defined borders:
    • With CSS, you have full control over the margins.
    • There are four properties for setting the margin for each side of an element (top, right, bottom, and left).
      • margin-top
      • margin-right
      • margin-bottom
      • margin-left
    • All the margin properties can have the following values:
      • auto - You can set the margin property to auto to horizontally center the element within its container.
      • length - You can specifies a margin in px, pt, cm, etc..
      • Percentage % - You can specifies a margin in % of the width of the containing element.
      • inherit - You can specifies that the margin should be inherited from the parent element.
    • MARGIN-TOP
      • Syntax:-
        • margin-top: 20px;
        • top margin is 20px
    • MARGIN-RIGHT
      • Syntax:-
        • margin-right: 20px;
        • right margin is 20px
    • MARGIN-BOTTOM
      • Syntax:-
        • margin-bottom: 20px;
        • bottom margin is 20px
    • MARGIN-LEFT
      • Syntax:-
        • margin-left: 20px;
        • left margin is 20px
  2. Margin-Shorthand Attributes:-
    • To shorten the code, it is possible to specify all the margin properties in one property.
    • If the margin property has four values:
      • Syntax:-

→ CSS Border

  1. Border-style Attributes:-
    • The border-style property specifies what kind of border to display.
    • Syntax:-

      1. p{border-style:dotted}

      2. p{border-style:dashed}

      3. p{border-style:solid}

      4. p{border-style:double}

      5. p{border-style:groove}

      6. p{border-style:ridge}

      7. p{border-style:inset}

      8. p{border-style:outset}

      9. p{border-style:none}

      10. p{border-style:hidden}

      11. p{border-style:dotted dashed solid double} means p{border-style:(top left bottom right)←border style}

  2. Border-width Attributes:-
    • The border-width property specifies the width of the four borders.
    • The width can be set as a specific size (in px, pt, cm, em, etc) or by using one of the three pre-defined values: thin, medium, or thick:
    • Syntax:-

      1. p{border-width:thick}

      2. p{border-width:medium}

      3. p{border-width:thin}

      4. p{border-width:15px}

  3. Border-color Attributes:-
    • The border-color property is used to set the color of the four borders.
    • The color can be set by:
      • name - specify a color name, like "red"
      • HEX - specify a HEX value, like "#ff0000"
      • RGB - specify a RGB value, like "rgb(255,0,0)"
      • HSL - specify a HSL value, like "hsl(0, 100%, 50%)"
      • transparent
    • If border-color is not set, it inherits the color of the element.
    • Syntax:-

      1. p{border-color:red}

      2. p{border-color:green}

      3. p{border-color:blue}

    • The border-color property can have from one to four values (for the top border, right border, bottom border, and the left border).
    • Syntax:-

      1. p{border-color: red green blue yellow;}

  4. Border-sides Attributes:-
    • In CSS, there are also properties for specifying each of the borders (top, right, bottom, and left):
    • If the border-style property has four values:
      • Syntax:-
        • border-style: dotted solid double dashed;
        • top border is dotted
        • right border is solid
        • bottom border is double
        • left border is dashed
    • If the border-style property has three values:
      • Syntax:-
        • border-style: dotted solid double;
        • top border is dotted
        • right and left borders are solid
        • bottom border is double
    • If the border-style property has two values:
      • Syntax:-
        • border-style: dotted solid;
        • top and bottom borders are dotted
        • right and left borders are solid
    • If the border-style property has one values:
      • Syntax:-
        • border-style: dotted;
        • all four borders are dotted
  5. Border-Shorthand Attributes:-
    • Like you saw in the previous page, there are many properties to consider when dealing with borders.
    • To shorten the code, it is also possible to specify all the individual border properties in one property.
    • The border property is a shorthand property for the following individual border properties:
      • border-width
      • border-style (its compulsory to display border)
      • border-color
    • Syntax:-

      1. p{border:5px solid red;}

      2. p{border-top:5px solid red;}

      3. p{border-left:5px solid red;}

      4. p{border-bottom:5px solid red;}

      5. p{border-right:5px solid red;}

  6. Border-Radius Attributes:-
    • The border-radius property is used to add rounded borders to an element:
    • Syntax:-

      1. p{border-radius:0px}

      2. p{border-radius:5px}

      3. p{border-radius:8px}

      4. p{border-radius:12px}

      5. p{border-radius:30px}

→ CSS PADDINGS

  1. Padding Attributes:-
    • The CSS padding properties are used to generate space around an element's content, inside of any defined borders.
    • With CSS, you have full control over the paddins.
    • There are four properties for setting the paddings for each side of an element (top, right, bottom, and left).
      • padding-top
      • padding-right
      • padding-bottom
      • padding-left
    • All the padding properties can have the following values:
      • length - You can specifies a padding in px, pt, cm, etc..
      • Percentage % - You can specifies a padding in % of the width of the containing element.
      • inherit - You can specifies that the padding should be inherited from the parent element.
    • PADDING-TOP
      • Syntax:-
        • padding-top: 20px;
        • top padding is 20px
    • PADDING-RIGHT
      • Syntax:-
        • padding-right: 20px;
        • right padding is 20px
    • PADDING-BOTTOM
      • Syntax:-
        • padding-bottom: 20px;
        • bottom padding is 20px
    • PADDING-LEFT
      • Syntax:-
        • padding-left: 20px;
        • left padding is 20px
  2. Padding-Shorthand Attributes:-
    • To shorten the code, it is possible to specify all the padding properties in one property.
    • If the padding property has four values:
      • Syntax:-
        • padding: 20px 15px 10px 5px;
        • top padding is 20px
        • right padding is 15px
        • bottom padding is 10px
        • left padding is 5px
    • If the padding property has three values:
      • Syntax:-
        • padding: 20px 15px 5px;
        • top padding is 20px
        • right and left paddings are 15px
        • bottom padding is 5px
    • If the padding property has two values:
      • Syntax:-
        • padding: 20px 10px;
        • top and bottom paddings are 20px
        • right and left paddings are 10px
    • If the padding property has one value:
      • Syntax:-
        • padding: 20px;
