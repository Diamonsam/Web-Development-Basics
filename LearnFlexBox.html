⇒ CSS Flexbox
⇒ CSS Flexbox Layout Module ⇒ Before the Flexbox Layout module, there were four layout modes:
  • Block, for sections in a webpage
  • Inline, for text
  • Table, for two-dimensional table data
  • Positioned, for explicit position of an element
⇒ The Flexible Box Layout Module, makes it easier to design flexible responsive layout structure without using float or positioning.
⇒ Flexbox Elements
⇒ To start using the Flexbox model, you need to first define a flex container.
⇒ Syntax For Defining a Flex Container :-
<div class="flex-container">
     <div class="flex-item"> Item 1 <div>
     <div class="flex-item"> Item 2 <div>
     <div class="flex-item"> Item 3 <div>

  • There are six different properties you can set for flex-items:
    1. order
    2. flex-grow
    3. flex-shrink
    4. flex-basis
    5. flex
    6. align-self
1 . The order Property
  • The order property specifies the order of the flex items.
  • The first flex item in the code does not have to appear as the first item in the layout.
  • The order value must be a number, → The default value is 0.
    • order:1; ⇒ The smaller order value element will be place first in the stacks .
      • Syntax :-
        <div class="flex-container" style="display:flex;">
             <div class="flex-item" style="order:2"> Item 1 <div>
             <div class="flex-item" style="order:3"> Item 2 <div>
             <div class="flex-item" style="order:1"> Item 3 <div>
        <div>
      • Result :-
        Item 1
        Item 2
        Item 3
2 . The flex-grow Property
  • The flex-grow property specifies how much a flex item will grow relative to the rest of the flex items.
  • The flex-grow value must be a number, → The default value is 0.
    • flex-grow:4; ⇒ It Means Make the flex item grow four times faster than the other flex items.
      • Syntax :-
        <div class="flex-container" style="display:flex;">
             <div class="flex-item" style="flex-grow:1"> Item 1 <div>
             <div class="flex-item" style="flex-grow:4"> Item 2 <div>
             <div class="flex-item" style="flex-grow:1"> Item 3 <div>
        <div>
      • Result :-
        Item 1 - grow 1
        Item 2 - grow 4 times faster
        Item 3 - grow 1
3 . The flex-shrink Property
  • The flex-shrink property specifies how much a flex item will shrink relative to the rest of the flex items.
  • The flex-shrink value must be a number, → The default value is 1.
    • flex-shrink:4; ⇒ It Means Make the flex item shrink four times faster than the other flex items.
      • Syntax :-
        <div class="flex-container" style="display:flex;">
             <div class="flex-item" style="flex-shrink:1"> Item 1 <div>
             <div class="flex-item" style="flex-shrink:4"> Item 2 <div>
             <div class="flex-item" style="flex-shrink:1"> Item 3 <div>
        <div>
      • Result :-
        Item 1 - shrink 1
        Item 2 - shrink 4 times faster
        Item 3 - shrink 1
4 . The flex-basis Property
  • The flex-basis property specifies the initial length of a flex item.
  • ⇒ When flex-direction: is set to row than flex-basis: will control element width.
  • ⇒ When flex-direction: is set to column than flex-basis: will control element height.
    • flex-basis Property With flex-direction:row; ⇒ It Means The flex-basis property will control the width of a flex item.
      • Syntax :-
        <div class="flex-container" style="display:flex; flex-direction:row;">
             <div class="flex-item" style="flex-basis:20vw"> Item 1 <div>
             <div class="flex-item" style="flex-basis:40vw"> Item 2 <div>
             <div class="flex-item" style="flex-basis:20vw"> Item 3 <div>
        <div>
      • Result :-
        Item 1 - 20 view width
        Item 2 - 40 view width
        Item 3 - 20 view width
    • flex-basis Property With flex-direction:column; ⇒ It Means The flex-basis property will control the height of a flex item.
      • Syntax :-
        <div class="flex-container" style="display:flex; flex-direction:column;">
             <div class="flex-item" style="flex-basis:5vh"> Item 1 <div>
             <div class="flex-item" style="flex-basis:10vh"> Item 2 <div>
             <div class="flex-item" style="flex-basis:5vh"> Item 3 <div>
        <div>
      • Result :-
        Item 1 - 5 view height
        Item 2 - 10 view height
        Item 3 - 5 view height
5 . The flex Property
  • The flex property is a shorthand property for setting the flex-grow, flex-shrink, and flex-basis properties.
    • flex: 2 3 400px; ⇒ It Means set the flex-grow to 2, flex-shrink to 3, and flex-basis to 400px.
      • Syntax :-
        <div class="flex-container" style="display:flex;">
             <div class="flex-item" style="flex:1 2 20vw"> Item 1 <div>
             <div class="flex-item" style="flex:2 4 40vw"> Item 2 <div>
             <div class="flex-item" style="flex:1 2 20vw"> Item 3 <div>
        <div>
      • Result :-
        Item 1 - grow 1 shrink 2 times faster width 20vw
        Item 2 - grow 2 shrink 4 times faster width 40vw
        Item 3 - grow 1 shrink 2 times faster width 20vw
6 . The align-self Property
  • The align-self property specifies the alignment for the selected item inside the flexible container.
  • The align-self property overrides the default alignment set by the container's align-items property.
  • There are 5 different values you can set for align-self:
    1. stretch → align-self: stretch; ⇒ The element is positioned to fit the container.→ This Is Default Value
    2. flex-start → align-self: flex-start; ⇒ The element is positioned at the beginning of the container.
    3. flex-end → align-self: flex-end; ⇒ The element is positioned at the end of the container.
    4. center → align-self: center; ⇒ The element is positioned at the center of the container.
    5. baseline → align-self: baseline; ⇒ The element is positioned at the baseline of the container.
    • Syntax :-
      <div class="flex-container" style="display:flex;">
           <div class="flex-item" style="align-self: stretch;">    Item 1 <div>
           <div class="flex-item" style="align-self: flex-start;"> Item 2 <div>
           <div class="flex-item" style="align-self: flex-end;">   Item 3 <div>
           <div class="flex-item" style="align-self: center;">     Item 4 <div>
           <div class="flex-item" style="align-self: baseline;">   Item 5 <div>
      <div>
    • Result :-
      Item 1 - align-self: stretch
      Item 2 - align-self: flex-start
      Item 3 - align-self: flex-end
      Item 4 - align-self: center
      Item 5 - align-self: baseline
