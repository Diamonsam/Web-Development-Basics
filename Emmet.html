→  To generate Tag with Class

Format :- tagname.classname + Enter (or) Tab key
For example div.container will create <div class="container"><div>
One classname you can use in multiple elements.

→  You can crete single element with multiple classes.

Format :- tagname.classname1.classsname2.classname3 + Enter (or) Tab key
For example div.container.redBg.greenText will create <div class="container redBg greenText"><div>

→  Default Element is Div when no element is specified.

Format :- .classname + Enter (or) Tab key
For example .container will create <div class="container"><div>


→  To generate Tag with Id

Format :- tagname#idname + Enter (or) Tab key
For example div#container will create <div id="container"><div>
One idname is unique & you can't use it in multiple elements, you can only use it in one element.

You can not crete single element with multiple Ids.

→  Default Element is Div when no element is specified.

Format :- #classname + Enter (or) Tab key
For example #container will create <div id="container"><div>


→  To generate Multiple Tags

Format :- tagname*number + Enter (or) Tab key
For example div*5 will create Five divs

→  To Go To New Line From Inside Tag

Mac Shortcut ⌘ + ⏎
Window Shortcut Ctrl + Enter

→  To Comment any line or selected text

Mac Shortcut ⌘ + /
Window Shortcut Ctrl + /

→  To Type In Multiple Lines

Mac Shortcut ⌘ + ⌥ + ↓ ←OR→ ⌘ + ⌥ + ↑
Window Shortcut Ctrl + Alt + Down-Arrow ←OR→ Ctrl + Alt + Up-Arrow

→  To Generate Dummy Text With (N) Number words

Write → lorem10 & Press Enter to Generate 10 Words Dummy Text.
For Example lorem10 = Lorem ipsum dolor sit amet, consectetur adipisicing elit. Veritatis, eos!

Write → lorem20 Press Enter to Generate 20 Words Dummy Text.
For Example lorem20 = Lorem ipsum dolor sit amet consectetur adipisicing elit. Porro at accusamus voluptatibus, magnam excepturi est ut dolores nesciunt vero ipsam.
